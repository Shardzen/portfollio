# Portfolio React

Ce projet est un portfolio personnel créé avec React, utilisant Tailwind CSS pour le style et Framer Motion pour les animations.

## Prérequis

Assurez-vous d'avoir Node.js et npm installés sur votre machine.

## Installation et Lancement

1. Naviguez vers le dossier du projet :
   ```
   cd portfollio
   ```

2. Installez les dépendances :
   ```
   npm install
   ```

3. Lancez l'application en mode développement :
   ```
   npm start
   ```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur pour voir le site.

Le site se rechargera automatiquement lorsque vous apportez des modifications au code.
